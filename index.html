ANT Book Club Meeting Notes

This page contains an archive of scribbling boards from our analytic number theory book club meetings. The reference textbook for these meetings as well as the boards listed below is Apostol, Introduction to Analytic Number Theory.

Note that these boards are not meant to be a systematic exposition of analytic number theory. Instead these are just a collection of some examples that illustrate some of the theorems in the reference textbook and some intermediate steps that are not explicitly expressed in the book. These boards were used to aid the discussions during our book club meetings. As a result, the content of these boards is informal in nature and is not intended to be a substitute for the book or the actual book club meetings.

If you would like to join our book club meetings, please go here.

If you find any mistakes in the content of the board files, please create a new issue or send a pull request.
